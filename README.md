# Mario & Luigi Encanadores

Site institucional temático para a empresa de encanamento Mario & Luigi, com formulário de contato animado e vídeo de fundo.

## Demonstração

Link do projeto: https://eduardoadf-dev.github.io/Projeto-M-rio-e-Luigi/

## Funcionalidades

- Vídeo de fundo com máscara gradiente
- Formulário de contato com animação de entrada
- Fecha o formulário com clique fora ou tecla `Escape`
- Feedback visual ao enviar o formulário
- Link direto para WhatsApp
- Cabeçalho fixo com navegação
- Responsivo para mobile

## Tecnologias

- HTML
- CSS
- JavaScript (vanilla)

## Como usar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Não requer instalação ou dependências
